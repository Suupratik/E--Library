📚 Library Management System (LMS) – React Project
Developed by: Suupratik Miitra
A fully functional, role-based Library Management System built with React.js, designed for efficient library operations, intuitive navigation, and responsive design.

🎯 Project Overview
This LMS is a modern web application that enables Admins, Teachers, and Students to manage, issue, and track books seamlessly.
It eliminates the inefficiencies of manual record-keeping by offering a centralized digital platform for all library activities.

🚀 Features
🔐 Authentication & Authorization
Admin Login – Full privileges to manage users and books.

Teacher Login – Issue, return, and check availability of books.

Student Login – Browse and request books.

📖 Book Management
Add, edit, and delete book entries.

Maintain detailed book records: title, author, category, and availability.

Manage cover images (optional feature).

🔍 Search & Filter
Instant search by title, author, or category.

Apply filters to find books by status or genre.

📊 Borrow & Return
Track borrowing records with due dates.

Restrict issuing of unavailable books.

Maintain borrowing history.

📱 Responsive UI
Works smoothly on desktops, tablets, and mobile devices.

Minimal, clean interface for easy use.
